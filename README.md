# Hallituskysely

Tämä projekti on interaktiivinen kysely, jossa käyttäjä voi testata tietämystään Suomen hallituskoalitioista. Ohjelma hakee Tilastokeskuksen JSON Stat 2 -dataa ja näyttää tulokset JavaScript-kaavioina.

## Ominaisuudet
- Kysely Suomen hallituskoalitioista
- Datan haku Tilastokeskuksen rajapinnasta
- Datan visualisointi kaavioina
- Lopputuloksen näyttäminen

## Käyttö
1. Avaa `index.html` selaimessa.
2. Vastaa kysymyksiin.
3. Katso tulokset.

## In English

This project is an interactive quiz about Finnish government coalitions. The program fetches data from Tilastokeskus (Statistics Finland) and displays results in JavaScript charts.

### Features
- Quiz about Finnish government coalitions
- Fetches Data from Tilastokeskus API
- Results visualized as charts
- User score at the end

### Usage
1. Open `index.html` in your browser.
2. Answer the questions.
3. View your results.
